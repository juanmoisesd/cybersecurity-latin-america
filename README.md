# Cybersecurity in Latin America (2018-2024)

[![Mendeley Data](https://img.shields.io/badge/Mendeley%20Data-h5swxzt3kn-red)](https://data.mendeley.com/drafts/h5swxzt3kn) [![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/) [![Files](https://img.shields.io/badge/Files-20-blue)]()

## Description
Cybersecurity incident data, infrastructure metrics and national digital readiness indicators for Latin America (2018-2024). Sources: ITU GCI 2020/2024, Kaspersky Security Network, IBM X-Force Threat Intelligence, IDB/OAS, national CERT/CSIRT logs.

**Mendeley Data:** [h5swxzt3kn](https://data.mendeley.com/drafts/h5swxzt3kn) | **Spanish version:** [ciberseguridad-america-latina](https://github.com/juanmoisesd/ciberseguridad-america-latina)

## Key Statistics (2024)
| Indicator | Value | Source |

| Ransomware growth 2018-2024 | +324% | IBM X-Force |
| Financial losses estimated (2024) | USD 6.2 billion | ITU/IDB |
